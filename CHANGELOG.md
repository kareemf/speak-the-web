# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-28

### Added

- Initial release of Speak the Web
- URL input for fetching and reading web articles
- Safari Share Extension for sharing directly from Safari
- Text-to-speech with iOS system voices (AVSpeechSynthesizer)
- Text-to-speech with Piper neural voices via sherpa-onnx
- Playback controls (play, pause, stop, skip forward/back)
- Seekable progress bar
- Speed controls (0.5x to 2x)
- Auto-generated table of contents from HTML headings
- Section navigation (jump to any heading)
- Voice selection from all available iOS voices
- Background audio playback
- Reading progress persistence
- Recent articles list
- "Start reading from here" text selection
